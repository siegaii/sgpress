## Natives modules
+ 当前层内容由JS实现
+ 提供程序直接调用库，fs path http等
+ js语言无法直接操作底层硬件设置

## 底层
+ v8：执行JS代码，提供桥梁接口
+ libuv：时间循环、时间队列，异步IO
+ 第三方模块：zlib、http、c-ares
